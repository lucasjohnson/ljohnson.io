## About

I'm a frontend engineer with 9+ years of experience building production web applications with React, TypeScript, and Next.js. I've worked across fintech, enterprise, and agency environments — always focused on shipping accessible, performant UIs that real users depend on.

Most recently at Candis, I built core product features for a fintech application serving thousands of businesses across Germany, co-owned a shared design system used by the entire engineering team, and drove testing practices with Cypress, Vitest, and Storybook. Before that, I delivered enterprise-scale projects at Monks for clients including Hewlett-Packard, Scotiabank, and Cineplex.

I care about code quality, accessibility (WCAG AA across every project I've worked on), and building UI that's both well-tested and a pleasure to use. I'm comfortable working closely with designers and product teams, and I do my best work in collaborative, agile environments.

Currently based in Berlin and open to new opportunities.

Tech: React, TypeScript, Next.js, GraphQL, Prisma, Vitest, Playwright, Cypress, Jest, Storybook, REST APIs, WCAG AA, ARIA, Claude Code
